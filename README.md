# OpenRedux

OpenRedux is a modern, open-source mod manager and graphics configurator designed for GTA V, RAGE MP, and FiveM. Built with Tauri, React, and TypeScript, it offers a sleek interface for managing and customizing game modifications.

## Features

- **Mod Library**: Browse and discover high-quality graphics mods, vehicle packs, and scripts.
- **One-Click Install**: Seamlessly install mods like "Network Redux V3" with automatic resource handling.
- **Real-time Configurator**: Fine-tune visual settings (Sky Brightness, Bloom, Volumetric Fog) and gameplay tweaks (Recoil, Ragdolls) on the fly.
- **Modern UI**: A responsive, dark-mode-first interface built with Tailwind CSS and Phosphor Icons.
- **Performance Focused**: Lightweight application powered by Rust and Tauri.

## Tech Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS
- **Icons**: Phosphor Icons
- **Backend/Core**: Tauri (Rust)

## Getting Started

### Prerequisites

- Node.js (v16+)
- Rust (latest stable)

### Installation

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run tauri dev
   ```
